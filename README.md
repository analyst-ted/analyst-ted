## 🚀 Featured Projects

### 🏥 Hospital Readmission Prediction
> Predicts whether a diabetic patient will be readmitted 
> within 30 days of discharge — enabling early intervention

**Key Results:**
- 53% Recall on 11.2% minority class after threshold optimization
- Identified elderly patients (70-90) as highest risk group
- Engineered ICD-9 diagnosis codes from 700+ categories to 18
- Handled 88.8/11.2 class imbalance using SMOTE

**Tech:** XGBoost · Scikit-learn · SMOTE · Streamlit · Python

### 🏦 Loan Default Risk Engine
> Deep learning pipeline predicting loan default risk on 
> 395,000+ LendingClub applications

**Key Results:**
- 67.2% default recall — 13x improvement over baseline
- $143.7M in loan capital protected on test set alone
- Neural network with computed class weights outperformed
  logistic regression by 13x on minority class recall
- Full business impact quantified in dollar terms

**Tech:** TensorFlow/Keras · Scikit-learn · Streamlit · Python
